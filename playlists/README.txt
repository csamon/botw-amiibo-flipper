==============================================================
  PLAYLISTS AMIIBO - ZELDA BREATH OF THE WILD - FLIPPER ZERO
  Chaque amiibo est dans UNE SEULE playlist selon sa meilleure utilité
==============================================================

ORDRE DE SCAN QUOTIDIEN RECOMMANDÉ :
  1. ZELDA-TOP          (obligatoire chaque jour)
  2. ZELDA-EQUIPEMENTS  (tant que tu n'as pas tous les sets)
  3. ZELDA-POISSONS     (si tu veux cuisiner)
  4. ZELDA-VIANDES      (si tu veux cuisiner)
  5. ZELDA-PLANTES      (si tu veux cuisiner)
  6. ZELDA-DIVERS       (fin de vie, pas urgent)

INSTALLATION :
  Copie FlipperAmiibo-main/ vers /ext/nfc/ sur la SD du Flipper
  => Structure : /ext/nfc/Legend_of_Zelda/... et /ext/nfc/Super_Smash_Bros/...
  Copie aussi ce dossier _PLAYLISTS_BOTW/ dans /ext/nfc/
  Utilise l'app "NFC Playlist" sur le Flipper Zero

RÈGLES BOTW :
  - 1 scan par amiibo par jour (reset minuit)
  - Astuce : couper internet Switch > Paramètres > Heure > avancer 24h = reset immédiat
  - Sauvegarde AVANT chaque scan, recharge si mauvais drop
  - 5 scans consécutifs sans rare = 6ème scan garanti rare (pity system)
  - Besoin du Parapente pour drops complets
  - Besoin d'1 Bête Divine libérée pour items exclusifs (sets d'armure, etc.)

==============================================================
  ZELDA-TOP.txt  (14 fichiers)
==============================================================
Les incontournables. Items exclusifs et uniques, introuvables ailleurs.
Scanner en PRIORITÉ ABSOLUE chaque jour.

  SSB Zelda (3 fichiers, 3 UIDs différents = 3 chances/jour !)
    => Arc Crépuscule : meilleur arc du jeu, portée infinie, flèches lumière illimitées
    => Sinon : Royal Bow, Knight's Bow

  SSB Link (3 fichiers = 3 chances/jour)
    => Epona au 1er scan SEULEMENT - enregistre-la immédiatement à une écurie !
       (stats max : Force 5 / Vitesse 5 / Endurance 5, jamais obtainable autrement)
    => Sinon : boucliers et épées Knight/Royal

  Wolf Link MAX LEVEL + Wolf Link 20 Hearts
    => Invocable PLUSIEURS FOIS par jour (pas limité comme les autres !)
    => Compagnon qui chasse, attaque les ennemis, détecte les coffres
    => Wolf_Link_MAX_LEVEL en priorité, puis 20_Heart si besoin

  Guardian BotW
    => Flèches Anciennes (one-shot les Guardians et la plupart des boss)
    => Seule source renouvelable quotidienne de Flèches Anciennes
    => Aussi : Armes Guardian, Ancient Cores (craft armure ancienne)

  Link Majora's Mask (30th Anniversary)
    => Set Fierce Deity (meilleure armure d'attaque du jeu = même bonus que Barbarian Set)
    => Épée Fierce Deity exclusive

  Les 4 Champions (Daruk / Mipha / Urbosa / Rivali)
    => Casques Divins exclusifs : Daruk=Rudania, Mipha=Ruta, Urbosa=Naboris, Rivali=Medoh
    => Portent les HP des ennemis (comme la Tunique du Champion)
    => Avec l'Armure Ancienne complète (2 étoiles) = bonus Ancient Proficiency (+80% dégâts)
    => Bonus loose : Urbosa/Daruk = viande prime/gourmet | Mipha = poisson | Rivali = flèches

==============================================================

==============================================================
Sets d'armures et armes exclusifs, mais priorité moindre que TOP.
Scanner tant que tu n'as pas tout collecté.

  Link Twilight Princess    => Set Crépuscule (bonus : Master Sword Beam Up)
  Link Skyward Sword        => Set du Ciel + Épée de la Déesse (loose : volaille)
  Ocarina of Time Link      => Set du Temps + Épée Biggoron (loose : viande gourmet)
  Wind Waker Link           => Set des Vents + Boomerang Brise-mer (loose : poisson)
  Sheik x2 (BOTW_SSB + SSB) => Masque de Sheik (vitesse nocturne)
  Ganondorf x2              => Armes Royal Claymore / Knight's Claymore régulières
  8-Bit Link                => Set du Héros (bonus : Master Sword Beam Up)
  Young Link x2 (Yong+SSB)  => Set du Temps variante (même pool que OoT Link)
  Link Archer BotW          => Royal Bow / Knight's Bow (loose : mix poisson+volaille)
  Link Rider BotW           => Selle du Voyageur + Bride du Voyageur (cosmétique cheval)
